# IoT_website
 
## Setup

1. Go to the root directory(Iot_website)

2. $ . venv/bin/activate

3. $ export FLASK_APP=run.py

4. $ flask run

5. go to http://127.0.0.1:5000/

Enjoy it!

## To exit the virtual environment

$ deactivate

## To interrupt the service

Control + C
